Hello, and thanks for reading.

This is an **example issue**. Write your newsletter in plain Markdown below the
`--- frontmatter ---` block above — the `subject:` becomes the email subject and
the optional `preheader:` is the grey preview line inboxes show next to it.

Markdown works as you'd expect:

- **bold**, *italic*, and [links](https://awonderfullife.ca/)
- bullet lists and numbered lists
- `> ` blockquotes and `##` headings

When you're happy with it:

1. Preview to yourself: `npm run send -- issues/<file>.md --test you@example.com`
2. Broadcast to the list: `npm run send -- issues/<file>.md`

Every email automatically gets the masthead, your content, and a per‑subscriber
unsubscribe link in the footer.
